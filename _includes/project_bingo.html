Project Summary
Live 1 person bingo game using socket.io. Make sure that you have two browser windows open for User and Admin screens...

  • API Requests: axios
  • API Backend: Node, Express
  • UI Backend: webpack, babel
  • API Routing: Express
  • UI Routing: React Router
  • API Model: View & Controller
  • Eslint Rules: Airbnb
  • Please run 1. npm install 2. npm run dev
